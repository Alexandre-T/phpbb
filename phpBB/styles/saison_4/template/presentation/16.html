Vous vous installez confortablement devant une machine à sous. Vos doigts se posent sur le tabouret galbé d'un cuir synthétique. L'assise est confortable. Tout est fait pour que vous vous sentiez à votre aise. Un porte-manteau se trouve à côté de chaque machine pour que vous vous installiez encore mieux. L'hôtel veut que vous dépensiez votre argent. Cela doit bien marcher, car les jetons défilent et, d'ailleurs, vous ne gagnez rien. Pas même une petite victoire avec deux trois jetons ! Rien ! Votre réserve de jetons diminue, mais vous poursuivez toujours avec cet espoir de gagner quelques jetons. Financièrement, vous n'en avez d'ailleurs pas forcément un grand besoin. C’est étonnant comme autour de vous le monde a disparu. Le jeu est captivant, mais les gains restent nuls.

Actions

  • Si vous décidez de garder une petite réserve de jetons dans vos poches, rendez-vous en 33.
  • Si vous décidez de tenter votre chance jusqu’au bout, rendez-vous en 40.
